Modularity

Modularity was achieved by using XMLHttpRequests and was implemented by Magne. This method was chosen because it will be easy to integrate with a back end at at later point in time. The header and footer is stored in separate files as indicated in the organizational scheme. The same method was used to retrieve data from json files. When this project is finished, the json data files will be replaced by a back end, using PostgreSQL and PHP, that essentially sends data in the same json format.
